 Changes Made
During development, the following changes were made:

1. *Navigation*
   - Added responsive *hamburger menu* for mobile screens.  
   - Highlighted active page link.  

2. *Styling*
   - Applied a *dark theme* with highlight color #1abc9c.  
   - Created consistent *card layouts* for services, gallery, and work sections.  
   - Redesigned *forms* for contact and involvement pages.  

3. *Social Media*
   - Added *Font Awesome icons* for Facebook, WhatsApp, Telegram, and Email.  
   - Ensured visibility of icons across *all pages*.  

4. *Branding*
   - Introduced a *logo* (small image + text).  
   - Consistent branding in header and footer.  

5. *Maps*
   - Embedded *Google Maps iframe* on Contact page.  
   - Styled with CSS for responsive display and rounded corners.  

6. *Responsiveness*
   - Implemented media queries for *mobile and tablet* layouts.
